VRM1.0 Viewer 使い方
操作手順を説明します。
準備
1
ファイルを配置
必要なVRMと関連ファイルを同一フォルダに配置します。
指定するフォルダ/
├ character.vrm
├ anime1.vrma
├ anime2.vrma
├ anime3.vrma
└ config.json
ファイル名は完全に同じである必要はありません。
拡張子が一致していれば読み込みます。
2
JSONファイルを準備
以下の構成にしてください。
[
  {
     "background": "#00b148"
  },
  {
    "vrma": "anime1.vrma",
    "exp": { "happy": 0.8 }
  },
  {
    "vrma": "anime2.vrma",
    "exp": { "angry": 1.0, "blink": 0.5 }
  },
  {
    "vrma": "anime3.vrma",
    "exp": {}
  },
  {
    "vrma": "",
    "exp": {}
  }
]
ファイルの読み込み個数に制限はありませんが、多ければ多いほど読み込みに時間がかかります。
上記のサンプル通りで、vrmaファイルが無い場合は""だけにしてください。 expには、Expressionを指定してください。独自に作ったものも指定できます。0～1を指定してください。
3
Viewerを起動
Select Folderボタンをクリックして、ファイルを指定します。
